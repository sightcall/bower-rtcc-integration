RtccInt. Annotation

new Annotation(rtccObject, callObject, settings)

Allow the user to annotate inside an HTML element. The default element is the videobox.
Parameters:
Name Type Argument Description
rtccObject Rtcc The object with the connexion to the cloud
callObject RtccCall The call involved in the drawing
settings object <optional>
Properties
Name Type Argument Default Description
container DOMobject | jQueryObject <optional>
 videobox The container where the drawing will take place. The default right click menu will be disabled.
pointerSrc string <optional>
 a red pointer The relative URL to the pointer image
isShare boolean <optional>
 If true, the annotations will take place in the shared video. The share must have been started already. Mandatory for plugin and driver.
circles object <optional>
Properties
Name Type Argument Default Description
premium string <optional>
 "orange circle" The absolute URL to the circle of the premium user.
external string <optional>
 green circle The absolute URL to the circle of the external user.
colors object <optional>
Properties
Name Type Argument Default Description
premium colorRGBA <optional>
 [255,174,0,255](orange) The color drawn by the premium user.
external colorRGBA <optional>
 [114,255,0,255](green) The color drawn by the external user.

//draw on the videobox
var annotation;

  annotation = RtccInt.Annotation(rtcc, call);
  annotation.setMode(RtccInt.Annotation.modes.DRAW);
